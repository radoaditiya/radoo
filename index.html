Object Oriented Programming (OOP)

OOP adala paradigma yang berfokus pada penggunaan "objek" untuk mempresentasikan data dan perilaku. Alih alih memandang program sebagai serangkaian instruksi prosedural,oop memandang sebagai kumpulan objek yang saling berinteraksi.

Konsep dasar OOP

1. objek(object):

objek adalah instance dari sebuuah class, objek memiliki state (data/atribut) dan bavior (perolaku/metode)
2.class(kelas):
class adalah blueprint atau templete untuk membuat objek,class mendefinisikan struktur dan perilaku yang akan di miliki objek objek nya

Contoh objek

dalam dunia nyata: mobil,manusia,buku

dalam program: user,product,transaction

Contoh Class Car

class Car {
  String model;
  String color;

  Car(this.model, this.color);

  void start(){
    print("model $color");
  }
}

void main(){
  Car myCar = Car("inova reborn", "hitam");
  myCar.start();
}

Kesimpulan

Materi ini membahas dasar-dasar Object-Oriented Programming (OOP) di Dart. OOP adalah paradigma pemrograman yang berfokus pada objek, di mana objek memiliki atribut (data) dan metode (perilaku). Kelas berfungsi sebagai blueprint untuk membuat objek, sedangkan objek adalah instance dari kelas tersebut.Secara umum, OOP membantu membuat program lebih terstruktur, modular, mudah dikembangkan, dan mudah dipelihara, terutama untuk aplikasi yang kompleks.
